# Trade Balance Analyzer

[![PHP Version](https://img.shields.io/badge/php-%3E%3D7.4-blue.svg)](https://php.net)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

Инструмент для обнаружения дисбаланса в системе обмена предметами компьютерной игры. 
Позволяет найти последовательность обменов, которая позволяет игроку вернуться к исходному предмету с увеличением количества золота.

### Описание задачи

В игре существует рынок предметов, где игрок может обменять свой уникальный предмет на другой, доплачивая или получая золото. Система считается уязвимой (дисбаланс), если существует последовательность обменов, которая позволяет игроку:

- Вернуться к исходному предмету
- Увеличить количество золота

Такая последовательность называется **положительным циклом** (для игрока).

### Алгоритм решения

Программа использует **поиск в глубину (DFS)** для обнаружения циклов с отрицательной суммой весов в ориентированном графе, где:

- **Вершины** — предметы (1..n)
- **Рёбра** — возможные обмены
- **Вес ребра** — количество золота, которое отдаёт игрок (отрицательное значение = игрок получает золото)

**Ключевая идея:** цикл выгоден для игрока, если сумма весов рёбер в нём **отрицательна** (так как отрицательный вес = получение золота).

### Требования

- PHP 7.4 или выше
- Composer (для автозагрузки)

### Установка


# Клонируйте репозиторий
git clone https://github.com/yourusername/trade-balance-analyzer.git
cd trade-balance-analyzer

# Установите зависимости (только автозагрузчик)
composer install

### Запуск
# С передачей файла через перенаправление
php trade_analizer.php < examples/example1.txt

# Через cat (Linux/Mac)
cat examples/example1.txt | php trade_analyzer.php

# Через type (Windows)
type examples\example1.txt | php trade_analyzer.php

# С использованием composer-скриптов
composer example1

composer example2
